| Collection | Path |
|---|---|
| Domains | `/ovirt-engine/api/domains` |
| Users | `/ovirt-engine/api/users` |
| Groups | `/ovirt-engine/api/groups` |

`developer@internal`, `demo@internal` (password `secret`).
