# oVirt Engine contract packs

Generated by `python tools/ovirt_api_inventory/generate_packs.py`
(Make target: `make generate-packs`).

Each `contracts/ovirt/<series>/` contains:

| File | Role |
|---|---|
| `api.json` | Declared Engine operations |
| `manifest.json` | Series metadata and counts |
| `deltas.json` | Added / removed ops vs previous series |

Index: [`ovirt/index.json`](ovirt/index.json).  
Docs: [API versions](../docs/api-versions.md), [API coverage](../docs/api_coverage.md).
